Dark Mode

Toggled via data-theme="dark" on the <html> element. Persisted to localStorage('theme'). Falls back to prefers-color-scheme on first visit. All colors swap via CSS custom properties — no additional classes needed. 

/* Implementation */
document.documentElement.setAttribute('data-theme', theme);
localStorage.setItem('theme', theme);

Financial Data Conventions

  • All monetary values use font-mono tabular-nums for alignment
  • Currency formatted with toLocaleString('en-US', { style: 'currency', currency: 'USD' })
  • Summary cards show whole dollars (minimumFractionDigits: 0)
  • Trade list shows cents (minimumFractionDigits: 2)
  • Percentages show one decimal: 12.4%
  • Shares: whole numbers by default, fractional to 3 decimals for mutual funds
  • Gains colored var(--gain) with ▲ prefix
  • Losses colored var(--loss) with ▼ prefix
  • Live prices colored var(--live-blue)
  • Ticker symbols always uppercase, font-mono font-semibold
